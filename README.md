# Random Number Generator V1.0

## Project Overview
This is a simple Java console application that generates a random number within a user-defined range. The user enters a minimum and maximum number, and the system generates a random number between those values.

## Features
- Welcome screen
- Main menu
- Random number generation within a range
- Input validation
- Exit option

## How to Run
1. Open the project in NetBeans or another Java IDE.
2. Run the `Main.java` file.
3. Choose an option from the menu.
4. Enter the minimum and maximum values.
5. View the generated random number.
